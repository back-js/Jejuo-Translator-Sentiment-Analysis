# 제주어 번역 및 감정 분석기

  * python >= 3.6
  * NumPy >= 1.11.1
  * Fairseq
  * Sentencepiece

  * flask
  * Pytorch >= 1.4.0
  * sacremoses
  * indic_nlp_library
